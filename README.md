#### Сущность сотрудник должна содержать три поля:
1. id целочисленное
2. name строковое
3. role строковое
#### Сервис должен реализовать след. набор методов
1. получить список сотрудников
2. создать нового сотрудника

4. изменить данные сотрудника
5. удалить данные сотрудника
